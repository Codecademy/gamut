# Component decision index

One line per component: what it's for, and what to reach for instead if this isn't quite it. Scan this before writing any layout code — it's the fastest way to catch a `Box`/`FlexBox` rebuild before it happens.

| Component                                                            | Category                 | Use when                                                                                     | Not this — use instead                                                                                                                                         |
| -------------------------------------------------------------------- | ------------------------ | -------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `FillButton`                                                         | atoms                    | Primary/high-emphasis action                                                                 | `StrokeButton` for secondary, `CTAButton` for marketing-only                                                                                                   |
| `StrokeButton`                                                       | atoms                    | Secondary/outlined action                                                                    | `FillButton` for primary                                                                                                                                       |
| `TextButton`                                                         | atoms                    | Low-emphasis inline action                                                                   | `Anchor` for navigation, not an action                                                                                                                         |
| `IconButton`                                                         | atoms                    | Icon-only compact action (needs `tip`)                                                       | `FillButton`/`StrokeButton` if there's label text                                                                                                              |
| `CTAButton`                                                          | atoms                    | Marketing/high-visibility promo only                                                         | Any standard UI action — use `FillButton`                                                                                                                      |
| `Anchor`                                                             | typography               | Navigation link                                                                              | `TextButton` for an action, not navigation                                                                                                                     |
| `Text`                                                               | typography               | All text rendering                                                                           | Never raw `<p>`/`<span>`/`<h1-6>`                                                                                                                              |
| `Markdown`                                                           | organisms                | Rendering markdown content                                                                   | `Text` for plain strings                                                                                                                                       |
| `Box` / `FlexBox` / `GridBox`                                        | layouts                  | Generic layout with no purpose-built equivalent                                              | See `components/index.md` itself first — rows of data → `List`/`DataList`/`DataTable`; on/off → `Toggle`; nav/actions → `Menu`; status → `Alert`/`Badge`/`Tag` |
| `LayoutGrid`                                                         | layouts                  | Full-page 12-column section layout                                                           | `GridBox`/`FlexBox` for smaller local regions                                                                                                                  |
| `Menu` / `MenuItem`                                                  | molecules                | Navigation or action list                                                                    | `SelectDropdown` for form value selection                                                                                                                      |
| `Tabs`                                                               | molecules                | Switching between content panels in one surface                                              | Stacked sections that require scrolling to compare                                                                                                             |
| `Disclosure`                                                         | molecules                | A single isolated expandable section                                                         | `List`'s expandable-row pattern for two or more items                                                                                                          |
| `DataTable`                                                          | organisms                | Compare data across rows (metrics, scores, statuses); not individually selectable/expandable | `DataList` if rows need selection/expansion                                                                                                                    |
| `DataList`                                                           | organisms                | Item-focused rows users open/expand/select                                                   | `DataTable` if the goal is cross-row comparison                                                                                                                |
| `List` / `ListRow` / `ListCol`                                       | organisms                | Fully custom repeating rows `DataTable`/`DataList` can't accommodate                         | `DataTable`/`DataList` first — only drop to `List` when those don't fit                                                                                        |
| `Card`                                                               | atoms                    | A bounded content surface                                                                    | `Box` with manual border/shadow — always a `Card` reinvention                                                                                                  |
| `Dialog`                                                             | molecules                | Binary-choice prompt (Confirm/Cancel, image/description only)                                | `Modal` for multi-view or complex content                                                                                                                      |
| `Modal`                                                              | molecules                | Multi-view, complex content, custom layouts                                                  | `Dialog` for a simple confirm/cancel                                                                                                                           |
| `Drawer`                                                             | atoms                    | Slide-in surface, not a full dialog                                                          | `Modal`/`Dialog` if it needs full focus containment                                                                                                            |
| `Flyout`                                                             | molecules                | Contextual side panel                                                                        | `Popover` for a small anchored surface                                                                                                                         |
| `Popover` / `PopoverContainer`                                       | molecules/atoms          | Small anchored contextual surface                                                            | `Flyout` for a larger panel                                                                                                                                    |
| `ToolTip`                                                            | molecules                | Supplementary info tied to an element via `aria-describedby`                                 | `InfoTip` for standalone informational content                                                                                                                 |
| `InfoTip`                                                            | molecules                | Standalone informational disclosure                                                          | `ToolTip` for describing another element                                                                                                                       |
| `Coachmark`                                                          | molecules                | Onboarding/feature-highlight callout                                                         | `Popover`/`InfoTip` for non-onboarding content                                                                                                                 |
| `Alert`                                                              | molecules                | Inline status/feedback banner (error/success/warning)                                        | Colored `Text`/`Box` — always an `Alert` reinvention                                                                                                           |
| `Toast` / `Toaster`                                                  | molecules                | Transient notification                                                                       | `Alert` for persistent inline status                                                                                                                           |
| `Badge`                                                              | atoms                    | Status indicator                                                                             | `Tag` for categorization — see `components/buttons.md`-style cross-link in `Badge.prompt.md`                                                                   |
| `Tag`                                                                | atoms                    | Categorization/labeling                                                                      | `Badge` for status                                                                                                                                             |
| `ProgressBar`                                                        | atoms                    | Linear completion indicator                                                                  | `RadialProgress` for a circular/dial treatment — always consider both                                                                                          |
| `RadialProgress`                                                     | atoms                    | Circular/dial completion indicator                                                           | `ProgressBar` for a linear treatment                                                                                                                           |
| `Shimmer` / `Spinner`                                                | atoms                    | Loading state                                                                                | `FeatureShimmer` — that's for feature-highlighting, not loading                                                                                                |
| `Video`                                                              | molecules                | Any video playback                                                                           | Custom `<video>` or third-party embed — never build one                                                                                                        |
| `GridForm` / `ConnectedForm`                                         | organisms                | Any functional form (submit/validate/reset)                                                  | Bare atoms (`Input`, `Select`, etc.) — only for non-submitted standalone controls                                                                              |
| `Input` / `Select` / `Checkbox` / `Radio` / `TextArea` / `FormGroup` | atoms                    | Standalone non-submitted controls (live search, filters)                                     | `GridForm`/`ConnectedForm` for anything with a submit step                                                                                                     |
| `SelectDropdown`                                                     | atoms                    | Searchable/multi-select/enriched dropdown                                                    | `Select` for a plain option list — simpler and more accessible                                                                                                 |
| `DatePicker`                                                         | organisms                | Date input/selection                                                                         | Hand-built calendar UI — never                                                                                                                                 |
| `Toggle`                                                             | atoms                    | Standalone on/off control                                                                    | A styled `Checkbox` pretending to be a switch                                                                                                                  |
| `ColorMode` / `Background`                                           | components/color-mode.md | Dark/light regions                                                                           | Manual CSS color overrides for mode switching                                                                                                                  |
| `FocusTrap`                                                          | atoms                    | Custom overlay focus containment                                                             | Reimplementing focus containment by hand                                                                                                                       |

See `overview-components.md`'s Component Discovery section for the underlying pattern-to-component reasoning this table summarizes.

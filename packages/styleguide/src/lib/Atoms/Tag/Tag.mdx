## Usage

Use a tag to categorize content, represent user inputs, or guide users to related actions. Tags can either be dismissable individually, or read-only when paired with other controls. Use Tags in designs like Filters.

### When NOT to use:
- Primary actions - for performing critical or primary actions, use the Button component instead.
- Prominent navigation - for standalone, high-priority navigation (e.g., links to key pages or sections), use an Anchor link instead,

Use the Read-only tag to categorize or label content in a static, non-interactive way. This tag is ideal for simple categorization that doesn't require user interaction.

The deprecated `editorColors` object in `@codecademy/gamut-styles` is still exported for backwards compatibility. Prefer semantic platform editor tokens (for example `editor-keyword`) and `theme.colors` from <LinkTo id="Foundations/Theme/PlatformTheme">Platform Theme</LinkTo> for new work.

This table lists each legacy **name** and **hex**, then the **closest** matching palette key for `theme.colors` on the platform theme. Distances use the **resolved hex** map (`corePalette` plus `platformPalette`—the same values behind `platformTheme._tokens.colors`), because `platformTheme.colors` only holds `var(--color-…)` references after color modes are applied. Only opaque `#RRGGBB` values are considered (`rgba(...)` entries are skipped). Keys used only for Monaco chrome (`comment-*`, `indent-*`, `line-number-*`) are excluded so the match stays on general palette swatches.

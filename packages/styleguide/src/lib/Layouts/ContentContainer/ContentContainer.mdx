## Usage

Use ContentContainer twhen creating page layouts to contain content
within a maximum width, as well as centering that content. By default, it provides responsive padding according to the updated grid system (see figma link above).

### Best practices:

- Use for full-page layouts to contain content within a maximum width and center it.

### When NOT to use:

- `FlexBox`, `GridBox`, or `Box for containing smaller areas of the page.

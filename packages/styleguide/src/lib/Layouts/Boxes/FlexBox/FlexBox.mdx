For flex layouts we've added a specific flex <code>FlexBox</code> component that extends <LinkTo kind="Layouts/Boxes/Box">Box</LinkTo> with extra props for common flex configurations.

## Usage guidelines

- Use this component if you need specific flex behaviors that can only be used by `flex` and `inline-flex` elements.
- Do not use this component if you want access to flex child props like `flex` or `alignSelf` as most components will have access to these props directly!

## States

`FlexBox` comes with a few helpful boolean props to make configuration easier (in addition to those on `Box`!), these exist as sorthands to make common configurations of props less verbose:

```tsx
// display="inline-flex"
<FlexBox inline />

// flexWrap="wrap"
<FlexBox wrap />

// alignItems="center" justifyContent="center"
<FlexBox center />

// flexDirection="row"
<FlexBox row />

// flexDirection="column"
<FlexBox column />
```

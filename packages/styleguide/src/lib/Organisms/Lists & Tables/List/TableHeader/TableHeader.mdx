## Usage

`TableHeader` renders a sticky `thead` with one header row. Use it when `List` has `as="table"` so column titles sit above `ListRow` body rows.

Place `ListCol` cells **directly inside** `TableHeader` (not wrapped in `ListRow`). Mark header cells with `columnHeader` (and optional `aria-sort` for sortable tables). For semantics and props, see <LinkTo id="Organisms/Lists & Tables/List/ListCol">ListCol</LinkTo> (Header columns).

`TableHeader` receives `spacing`, `variant`, and `scrollable` from `List`, same as `ListRow` and `ListCol`, so the header stays visually aligned with the table body.

## Pro CourseCard

### Title

`title` will be "Course" for the `CourseCard` when `pro` is true.

### Certificate

`grantsCertificate` and `pro` must be true for the certificate info to show on the card.
We only show certificates for pro exclusive courses, since only pro users can do pro courses and only pro users can gain certificates.

<Canvas>
  <Story
    name="Pro CourseCard"
    args={{
      title: 'Building Interactive JavaScript Websites',

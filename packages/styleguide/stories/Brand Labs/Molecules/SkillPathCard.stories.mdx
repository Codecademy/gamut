For the `SkillPathCard` we've extended the <LinkTo kind="brand-labs-molecules-contentgroupbasecard">ContentGroupBaseCard</LinkTo> with extra props specific to skill paths.

Note: This card will need to be wrapped in an Anchor or Link in order to be clickable.

<Canvas>
  <Story name="SkillPathCard">
    {(args) => (
      <Box pb={8}>
        <SkillPathCard {...args} />
      </Box>
    )}
  </Story>
</Canvas>

<Props story="SkillPathCard" />

## SkillPathCard with Image

`isFullSize` must be true for image to show on card.

<Canvas>
  <Story name="SkillPathCard with Image">
    {(args) => (
      <Box pb={8}>
        <SkillPathCard {...args} isFullSize />
      </Box>
    )}
  </Story>
</Canvas>

## SkillPathCard with Enrollment Status

Unlike the old <LinkTo kind="brand-labs-molecules-curriculumcard--curriculum-card#career-path-description-variant">
CurriculumCard</LinkTo> designs, we show the same image for unenrolled, in progress, and completed full size skill path cards. We achieve this by stripping "-inProgress" and "-completed" from the imageUrl.

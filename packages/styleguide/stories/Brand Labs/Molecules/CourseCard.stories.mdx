## CourseCard with Certificate

`grantsCertificate` and `pro` must be true for certificate to show on card.
We only show certificates for pro exclusive courses, since only pro users can do pro courses and only pro users can gain certificates.

<Canvas>
  <Story name="CourseCard with Certificate">
    {(args) => <CourseCard {...args} pro grantsCertificate />}
  </Story>
</Canvas>

# LandingPageFeaturesSection

<StoryStatus status="volatile" />

A features section for use in landing pages created in Contentful. Images can be icons or full size via `isIcon` prop. Consists of an array of features and optional title, description, cta, cta href, and test id.

<Canvas>
  <Story name="WithImages">
    {(args) => <LandingPageFeaturesSection {...args} />}
  </Story>
</Canvas>

<Props story="WithImages" of={LandingPageFeaturesSection} />

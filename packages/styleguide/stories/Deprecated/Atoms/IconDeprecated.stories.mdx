# Icon

<StoryStatus status="deprecated" />

Legacy icons as defined in the `@codecademy/gamut/src`package's `Icon` export.

Please use non-deprecated `Icon`s from the `@codecademy/gamut-icons` package instead.

<Props of={Icon} />

## Usage

In order to use this component you will need to import it from a specific deprecated path. Since we want
to limit usage of this component going forward we have added eslint rules that will prevent you from
importing.

```tsx
import { Icon } from '@codecademy/gamut/dist/deprecated';

const MyComponent = () => <Icon name="close" />;
```

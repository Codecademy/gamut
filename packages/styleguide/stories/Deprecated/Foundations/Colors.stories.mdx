# Deprecated Colors

<StoryStatus status="deprecated" />

## Gamut

We used to have a very different color palette. Please do not use these
anymore.

If you come across a color on the site while updating an older area and see
that it uses one of these, try to replace it with the closest color from or
new palette.

We may delete these as we find that consumers do not use them.

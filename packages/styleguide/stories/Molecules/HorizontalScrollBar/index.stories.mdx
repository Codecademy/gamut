The `HorizontalScrollBar` can be used to have content scroll across the x-axis. Through use of the `IntersectionObserver` web api, this component will automatically set `aria-hidden` labels to `true` when content moves off the x-axis. This component also has controls that appear on `focus` events to assist users with navigating content through the keyboard.

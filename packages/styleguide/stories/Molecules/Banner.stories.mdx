## Full-Width Icon

An informative icon can be added to the left of the banner.

<Canvas>
  <Story name="Icon Banner" args={{ icon: <AlertIcon /> }}>
    {(args) => <Banner {...args} />}
  </Story>
</Canvas>

## Bordered

If your banner is on above similar content (?), add a BannerStyle to it.

<Canvas>
  <Story name="Bordered" args={{ displayStyle: BannerStyle.BorderBottom }}>
    {(args) => <Banner {...args} />}
  </Story>
</Canvas>

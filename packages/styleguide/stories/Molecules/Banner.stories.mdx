## Full-Width Icon

An informative icon can be added to the left of the banner.

<Canvas>
  <Story name="Icon Banner">
    {(args) => (
      <Banner icon={<AlertIcon />} onClose={action('Banner closed.')}>
        Some Banner content!
      </Banner>
    )}
  </Story>
</Canvas>

## Bordered

If your banner is on above similar content (?), add a BannerStyle to it.

<Canvas>
  <Story name="Bordered">
    {(args) => (
      <Banner
        displayStyle={BannerStyle.BorderBottom}
        onClose={action('Banner closed.')}
      >
        Some Banner content!
      </Banner>
    )}
  </Story>
</Canvas>

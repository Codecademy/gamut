## Curriculum Card with Beta Sticker

If `beta` is true, no other tag information will be rendered. In-progress and completed progress states will override the beta sticker.

<Canvas>
  <Story
    name="Curriculum Card Beta"
    args={{
      beta: true,
    }}
  >
    {(args) => <CurriculumCard {...args} />}
  </Story>
</Canvas>

## Curriculum Card with Alternative Subtitle

If `showAltSubtitle` is true, only the scope will render in the subtitle.

<Canvas>
  <Story
    name="Curriculum Card Alternative Subtitle"
    args={{
      showAltSubtitle: true,
      scope: {
        Lesson: 25,
        Quiz: 30,
        article: 100,
      },
    }}
  >
    {(args) => <CurriculumCard {...args} />}
  </Story>
</Canvas>

## Curriculum Card with Description

If `description` value is populated, the description will render underneath the subtitle.

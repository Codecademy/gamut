This component can be used when you need more information in order to perform an onClick action. An example use case is an "Add Workspace" button. In order for a new workspace to be created, we need to know the type of workspace the user wants to create. So we display the workspace types in a dropdown list.

The Dropdown button can be displayed as either a FillButton or StrokeButton depending on the buttonType prop. If not provided, it will default to 'fill'. Additionally, you can align the dropdown to either the left or right edge of the button. You can also provide an onClick prop which will be invoked when the button is clicked.

The list of items that will be displayed in the dropdown should be provided by the dropdownItems prop. This prop should be an array of object which contains the following keys: id, text, icon (optional), clickHandler (optional), href (optional).

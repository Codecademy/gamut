## Basic vs. Ellipsis

We have two types of pagination - `basic` and `ellipsis`. The `basic` type has number as well as forward and back buttons. The `ellipsis` type has an ellipsis button bfeature which allows users to quickly jump through page numbers by `chapterSize`.

<Canvas>
  <Story
    name="Basic"
    args={{
      chapterSize: 5,
      totalPages: 27,
      onChange: () => {},

    {Pagination.bind({})}
  </Story>
  <Story

## Pagination Navigation

When using pagination as navigation between linked pages, use the `navigation` prop. Your `onChange` function should include programmatic navigation.

<Canvas>
  <Story
    name="default size"
    args={{
      chapterSize: 5,
      totalPages: 27,
      type: 'ellipsis',
      navigation: true,
      onChange: (pageNumber) => {
        window.open(
          `https://giphy.com/search/hamtaro-page-${pageNumber}`,
          '_blank'
        );
      },
    }}
  >
    {Pagination.bind({})}
  </Story>
</Canvas>

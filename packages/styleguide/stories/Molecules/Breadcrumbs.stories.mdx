## Payload

A `payload` can be passed to each crumb, useful when arbitrary data is required in `onClick`.

`payload` can be any type, and the proper types will be inferred when referenced in `onClick` (even when multiple crumbs have different `payload` types)

<Canvas withSource="open">
  <Story name="Breadcrumbs with payload">
    <Breadcrumbs
      crumbs={[
        {
          title: 'Home',
          href: '.',
          payload: 'I can be anything!',
        },
        {
          title: 'Here',
        },
      ]}
      onClick={(e, crumb) => {
        e.preventDefault();
        alert(crumb.payload); // eslint-disable-line no-alert
      }}
    />
  </Story>
</Canvas>

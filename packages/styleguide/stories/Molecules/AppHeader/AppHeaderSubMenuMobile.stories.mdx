This component is used in the mobile version of the header (<LinkTo kind="Molecules/AppHeaderMobile" story="AppHeaderMobile">AppHeaderMobile</LinkTo>).

Similar to the desktop <LinkTo kind="Molecules/AppHeader" story="AppHeader">AppHeader</LinkTo>, when an AppHeaderItem has type "dropdown", clicking on it opens a dropdown menu; on smaller screens, clicking on an AppHeaderItem of type "dropdown" opens the <LinkTo kind="Molecules/AppHeaderSubMenuMobile" story="AppHeaderSubMenuMobile">AppHeaderSubMenuMobile</LinkTo> component. The component contains a title indicating the category of the menu and a list of links.

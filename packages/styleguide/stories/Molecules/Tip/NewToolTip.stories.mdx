A short tip triggered on hover and focus of an interactive element. Use a tooltip to further explain an element’s primary action, a disabled button state, or to display a label for an icon-only button.

When using the NewToolTip component, the `id` prop is required to associate the tooltip with the interactive element and the interactive element's `aria-describedby` prop must be set to the `id` of the tooltip.

When a Button is disabled with a tooltip, you should use the aria-disabled props instead of disabled prop so the user can still focus the button and view the tooltip.

        </NewToolTip>

## IconButtons and ToolTips

When using an icon-only button, a tooltip displaying the label is required to provide context for the button's action. This is accessible directly through the `IconButton`'s required `tip` prop. You can adjust positioning and alignment of the tooltip by using the tipProps prop.

    <FlexBox center py={64} m={24}>
        <IconButton tip="Beautify" icon={SparkleIcon} tipProps={{alignment: "bottom-center"}}/>
        </FlexBox>
          )}

ContentContainer can be used when creating page layouts to contain content
within the `$grid-xl` maximum width of [1440px](/?path=/docs/foundations-layout--page#screensizes-and-media-queries), as well as centering that content and providing
responsive padding at our different breakpoints.

Use `ContentContainer` to keep page _content_ at a consistent maximum content width across various screen sizes and zooms.
See [the Student Center](https://codecademy.com/student-center) for an example.

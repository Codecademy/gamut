ContentContainer can be used when creating page layouts to contain content
within a maximum width, as well as centering that content. By default, it provides responsive padding according to the updated grid system (see figma link above).

Note: AppBar (used in Global Header) and Footer also use ContentContainer.
Check with the design team if different padding/margin is required on a page layout as additional spacing may cause misalignment with the Header and Footer.

# HiddenText

<StoryStatus status="stable" />

Visually hidden text that should still be narrated by screenreaders.

This is useful for cases where we visually indicate something not easily conveyed through semantic HTML.
Ideally those cases should be exceedingly rare; unfortunately, some of our older pages aren't set up well for this.

See https://wdn.unl.edu/page-title-h1-best-practices for more information.

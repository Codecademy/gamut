# SplitInterstitial

<StoryStatus status="stable" />

Large, 50% split content area used for information that has two facets,
such as for a payment page with an advertisement on one side and payment
form on another.

<Preview>
  <Story name="SplitInterstitial">
    <SplitInterstitial
      left={{
        children: (
          <React.Fragment>
            <Heading as="h1">Important stuff!</Heading>
            <Text>Some sort of information here, perhaps?</Text>
          </React.Fragment>
        ),
      }}
      right={{
        children: (
          <React.Fragment>
            <Heading as="h2">Less important...</Heading>
            <Text>Some more information here, maybe?</Text>
          </React.Fragment>
        ),
      }}
    />
  </Story>
</Preview>

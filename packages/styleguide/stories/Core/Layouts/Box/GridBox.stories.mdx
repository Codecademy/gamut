# GridBox

<StoryStatus status="volatile" />

For flex layouts we've added a specific flex `GridBox` component that extends `Box` with added flex props.

<Canvas>
  <Story
    name="GridBox"
    args={{
      children: 'Hello world I am a GridBox',
    }}
  >
    {(args) => {
      const elProps = {
        borderWidth: '1px',
        borderStyle: 'solid',
        borderColor: 'navy',
        padding: 16,
      };
      return (
        <GridBox {...args}>
          <Box {...elProps}>Child Element 1</Box>
          <Box {...elProps}>Child Element 2</Box>
        </GridBox>
      );
    }}
  </Story>
</Canvas>

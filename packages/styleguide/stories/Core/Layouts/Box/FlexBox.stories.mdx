# FlexBox

<StoryStatus status="volatile" />

For flex layouts we've added a specific flex `FlexBox` component that extends `Box` with added flex props.

<Canvas>
  <Story
    name="FlexBox"
    args={{
      children: 'Hello world I am a FlexBox',
    }}
  >
    {(args) => {
      const elProps = {
        borderWidth: '1px',
        borderStyle: 'solid',
        borderColor: 'navy',
        padding: 16,
      };
      return (
        <FlexBox {...args}>
          <Box {...elProps}>Child Element 1</Box>
          <Box {...elProps}>Child Element 2</Box>
        </FlexBox>
      );
    }}
  </Story>
</Canvas>

## Heading

This is a single primitive to handle all `h1-h6` tags
with defined size intervals to allow for semantic html that also
matches the designs.

Note: Please only [increase heading levels by 1](https://dequeuniversity.com/rules/axe/3.0/heading-order)
(and picking the appropriate size to match the design).

<Preview>
  <Story name="Heading">
    <LayoutGrid columnGap="md" rowGap="md">
      {headingSizes.map((size) => (
        <React.Fragment key={size}>
          <Column size={1}>
            <Container align="center">
              <Heading
                fontSize="sm"
                as="h2"
                hideMargin
                style={{ color: colors.gray[400] }}
              >
                {size}
              </Heading>
            </Container>
          </Column>
          <Column size={11}>
            <Container key={size} align="center">
              <Heading fontSize={size} as="h3" hideMargin>
                Heading
              </Heading>
            </Container>
          </Column>
        </React.Fragment>
      ))}
    </LayoutGrid>
  </Story>
</Preview>

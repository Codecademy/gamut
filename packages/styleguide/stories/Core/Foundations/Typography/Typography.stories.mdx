# Typography

<StoryStatus status="volatile" />

This page is meant as a reference for Typography variables and scales that are available
through both **Typescript** and **SCSS** variables in [`@codecademy/gamut-styles`](https://www.npmjs.com/package/@codecademy/gamut-styles). We heavily recommend
you use the <LinkTo kind="Typography" story="Heading">`<Heading />`</LinkTo> and <LinkTo kind="Typography" story="Text">`<Text />`</LinkTo>
primitives that we provide for your needs. They are built using these standards, so you should not need to recreate them.

## Font Stacks

We have 4 main font-stacks that are used through out our front end apps. They are roughly separated by functionality.

1. `Base` for all body text.
2. `Heading` for all headings and subheadings `h1`-`h6`
3. `Monospace` for code and other space-sensitive or themed content.
4. `System` largely unused but still defined for when system fonts are necessary.

We're about to swap out the font stacks used in `Base` and `Heading`.
You can use the new stacks ahead of time with:

1. `Apercu` will replace the majority of both body text and headings.
2. `Suisse` will be an occasional "accent" stack.

<Canvas withSource="none" isColumn>
  {Object.entries(DesignTokens.fontFamily).map(([alias, value]) => (
    <ScaleRow key={alias} hasAlias>
      <ScaleColumn>{alias}</ScaleColumn>
      <Text family={value}>{value.split(',')[0].replace(/"/g, '')}</Text>
    </ScaleRow>
  ))}
</Canvas>

### Usage

```tsx
import { fontFamily } from '@codecademy/gamut-styles';

fontFamily.base; // base | heading | monospace | system
```

```scss
@import '@codecademy/gamut-styles/utils';

.text {
  font-family: $font-family-base; // $font-family-base | $font-monospace | $font-system
}
```

## Font Sizes

We use a single scale for font sizes that are aliased to their most appropriate use case.

<Canvas withSource="none" isColumn>
  {Object.entries(DesignTokens.fontSize).map(([key, size]) => {
    return (
      <ScaleRow key={key} hasAlias>
        <ScaleColumn>{key}</ScaleColumn>
        <Text size={size}>{size}</Text>
      </ScaleRow>
    );
  })}
</Canvas>

### Usage

```tsx
import { fontSize } from '@codecademy/gamut-styles';

fontSize['p-small']; // p-body | p-large | title-1 | title-2 | title-2 | title-3 | title-4 | title-5 | title-6
```

```scss
@import '@codecademy/gamut-styles/utils';

.heading {
  font-size: $font-size-h1; // h1 - h6
}
.body {
  font-size: $font-size-base; // small | base
}
```

## Font Weights

We support two basic `font-weight` values for both body text `400` and headings `700`, while lighter or bolder variations may be
available for some font stacks, these will be the safest for their desired effects.

```tsx
import { fontWeight } from '@codecademy/gamut-styles';
```


## Font Stacks

We have 4 main font-stacks that are available through both **Typescript** and **SCSS** variables in`@codecademy/gamut-styles`

## Accessing these variables

Here is how to access and use them in TS:

### TS

```tsx
import { fontFamily } from '@codecademy/gamut-styles';

const Text = ({ children }) => (
  <p style={{ fontFamily: fontFamily.base }}>{children}</p>
);
```

#### SCSS

```scss
@import '@codecademy/gamut-styles/utils';

.base {
  font-family: $font-family-base;
}

.heading {
  font-family: $font-family-headings;
}

.monospace {
  font-family: $font-monospace;
}

.system {
  font-family: $font-system;
}
```

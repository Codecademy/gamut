## Font Sizes

We have 2 font scales. One for headings and one for body text.

### Usage

```tsx
import { fontSize } from '@codecademy/gamut-styles';

fontSize.heading.md; // xs | sm | md | lg | xl | xxl
fontSize.text.md; // sm | md | lg
```

```scss
@import '@codecademy/gamut-styles/utils';

.heading {
  font-size: $font-size-h1; // h1 - h6
}
.body {
  font-size: $font-size-base; // $font-size-small
}
```

## Font Stacks

We have 4 main font-stacks that are available through both **Typescript** and **SCSS** variables in`@codecademy/gamut-styles`

### Usage

```tsx
import { fontStack } from '@codecademy/gamut-styles';

fontStack.base; // base | heading | monospace | system
```

```scss
@import '@codecademy/gamut-styles/utils';

.text {
  font-family: $font-family-base; // $font-family-base | $font-family-headings | $font-monospace | $font-system
}
```

## Font Weights

We support two basic `font-weight` values for both body text and headings.

### Usage

```tsx
import { fontWeight } from '@codecademy/gamut-styles';

fontWeight.base; // base | heading
```

```scss
@import '@codecademy/gamut-styles/utils';

.text {
  font-weight: $font-weight-base; // $font-weight-base | $font-weight-headings
}
```

## Line Heights

### Usage

We support two basic `line-height` values for both body text and headings.

```tsx
import { lineHeight } from '@codecademy/gamut-styles';

lineHeight.text; // text | heading
```

```scss
@import '@codecademy/gamut-styles/utils';

.text {
  line-height: $line-height-base; // $line-height-headings | $line-height-base
}
```

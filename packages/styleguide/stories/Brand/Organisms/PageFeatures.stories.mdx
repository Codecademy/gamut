## Featured Info

Display text-only featurettes with no media associated.

<Canvas>
  <Story name="Featured Info">{(args) => <PageFeatures {...args} />}</Story>
</Canvas>

<PropsTable story="Featured Info" />

## Featured Images

In addition to the title and description, you can provide an image by setting the `featuresMedia` prop to `image`.

<Canvas>
  <Story name="Featured Images" args={{ featuresMedia: 'image' }}>
    {(args) => <PageFeatures {...args} />}
  </Story>
</Canvas>

## Featured Icons

Small icons can be used instead of full size images by setting the `featuresMedia` prop to `icon`.

<Canvas>
  <Story name="Featured Icons" args={{ featuresMedia: 'icon' }}>
    {(args) => <PageFeatures {...args} />}
  </Story>
</Canvas>

## Featured Stats

Promote a single stat by setting the `featuresMedia` prop to `stat` and providing `statText`.

<Canvas>
  <Story name="Featured Stats" args={{ featuresMedia: 'stat' }}>
    {(args) => <PageFeatures {...args} />}
  </Story>
</Canvas>

## Grid layout

If you specify a `maxCols` the features at desktop will wrap at that number of columns.

<Canvas>
  <Story name="Features Grid" args={{ maxCols: 3, featuresMedia: 'icon' }}>
    {(args) => <PageFeatures {...args} />}
  </Story>
</Canvas>

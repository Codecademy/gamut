## Horizontal

Testimonials default to the horizontal layout.

  <Story name="Testimonial">{(args) => <Testimonial {...args} />}</Story>

## Color modes

Testimonials can be displayed in both light and dark color modes.

<LayoutGrid gap={32}>
<Column size={6}>

### Light mode

<Story name="Light Mode">{(args) => <Testimonial {...args} />}</Story>

</Column>
<Column size={6}>

### Dark mode

<Story name="Dark Mode">
  {(args) => <Testimonial hideAvatar mode="dark" {...args} />}
</Story>

</Column>
<Column display="block"  size={6}>

### Vertical Variant

Testimonial can use the vertical layout for smaller containers. Testimonial will also switch to this variant for mobile viewports.

<Story name="Small Light">
  {(args) => <Testimonial variant="vertical" {...args} />}
</Story>

</Column>
<Column display="block" flexDirection="column" size={6}>

### Hidden Picture

Testimonial will hide the picture automatically if the `imageUrl` prop is `undefined`. It can also be hidden via the `hidePhoto` prop

<Story name="Medium Dark">
  {(args) => <Testimonial {...args} variant="horizontal" hidePhoto />}
</Story>

</Column>
</LayoutGrid>

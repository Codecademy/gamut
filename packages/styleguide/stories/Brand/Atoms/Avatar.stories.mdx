## Usage

```tsx
import { Avatar } from '@codecademy/gamut-labs';

<Avatar size={{ _: 32, sm: 64 }} />;
```

## Sizes

We have several options for avatar sizes (118, 98, 64, 48, 32) which you may configure with the responsive syntax. While we do not allow arbitrary sizes for Avatars if these sizes do not fit your use suggestions are welcome!

<Canvas>
  <Story name="Sizes">
    {(args) => (
      <FlexBox gap={24} alignItems="flex-end">
        {[118, 98, 64, 48, 32].map((size) => (
          <FlexBox column gap={16}>
            <Avatar {...args} size={size} />
            <FlexBox fontFamily="accent" center>
              {size}px
            </FlexBox>
          </FlexBox>
        ))}
      </FlexBox>
    )}
  </Story>
</Canvas>

### SelectDropdown

This is a variant of our `Select` component with a styled dropdown menu. `SelectDropdown` has a larger JavaScript dependency size than our standard `Select` component so should not be used unless specified in designs.

### JSXElement Options

Our `SelectDropdown` can show JSX Elements - neat! These should always be passed in as React Fragments and passed into options as an `object`, with the JSX Element as the value of the key/value pair like so -

Do not wrap these elements in a `<Box>` or `<FlexBox>`.

<Canvas>
  <Story
    name="JSXElementDropdown"
    args={{
      options: {
        hi: (
          <>
            <MiniStarIcon color="pink" />
            <Text>'hello'</Text>
          </>
        ),
        hey: 'no worries',
        updog: 'its chill',
      },
      placeholder: 'all about that',
    }}
  >
    {(args) => (
      <FlexBox width="10rem" height="25rem">
        <SelectDropdown {...args} />
      </FlexBox>
    )}
  </Story>
</Canvas>

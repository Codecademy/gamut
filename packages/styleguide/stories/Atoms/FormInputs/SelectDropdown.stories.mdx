### SelectDropdown

`SelectDropdown` has a larger JavaScript dependency size than our standard `Select` component so should not be used unless specified in designs. If you wish to define an intial value, you must format `SelectDropdown` as a [controlled component](https://reactjs.org/docs/forms.html#controlled-components).

<Canvas>
  <Story
    name="BaseDropdown"
    args={{
      options: ['Based', 'Regular Size', 'Normal'],
      placeholder: 'all about that',
    }}
  >
    {(args) => (
      <Box width="10rem" height="25rem">
        <SelectDropdown {...args} />
      </Box>
    )}
  </Story>
  <Story
    name="DisabledDropdown"
    args={{
      options: ['Disabled'],
      disabled: true,
      placeholder: 'Disabled',

    {(args) => (
      <Box width="10rem" height="25rem">

### Form Group + Select

`SelectDropdown` formats nicely in FormGroups and reflects error states.

<Canvas>
  <Story
    name="FormGroupStyled"
    args={{
      options: ['hello', 'hi', 'howdy'],
      value: 'oh no',
    }}
  >
    {(args) => (
      <Box width="25rem" height="25rem">
        <FormGroup label="i am big label" labelSize="large">
          <SelectDropdown {...args} />
        </FormGroup>
      </Box>
    )}
  </Story>
  <Story
    name="FormGroupStyledError"
    args={{
      options: ['Error', 'oh no', ':('],
      placeholder: 'cry cry cry',
    }}
  >
    {(args) => (
      <Box width="25rem" height="25rem">
        <FormGroup
          label="i am ~styled... but something is wrong"
          error="error message!! no :'("
        >
          <SelectDropdown error {...args} />
        </FormGroup>
      </Box>
    )}
  </Story>
</Canvas>

### Custom inputProps

Some of our forms require special props be passed directly onto the `<input/>` component. We can do that by passing an object, formatted `{propName: value}`, into our inputProps prop.

<Canvas>
  <Story
    name="CustomInputProps"
    args={{
      options: ['inspect me to see my inputProps', 'yes I am!', ':)'],
      inputProps: { hello: 'updog' },
      name: 'updog',
    }}
  >
    {(args) => (
      <Box width="25rem" height="25rem">
        <FormGroup label="i am ~styled">
          <SelectDropdown {...args} />
        </FormGroup>
      </Box>
    )}
  </Story>
</Canvas>

## Usage

```tsx
import { Menu, MenuItem, MenuSeparator } from '@codecademy/gamut';

const MyMenu = () => (
  <Menu variant="action">
    <MenuItem>Item 1</MenuItem>
    <MenuSeparator />
    <MenuItem href="cool">Link 1</MenuItem>
  </MenuSeparator>
);
```

## Variants

<LayoutGrid gap={32} py={24}>
<Column size={8} variant={false}>

### Select

Select menus are used for select dropdowns and non-navigation actions like code-editor actions. Examples of this are in `SelectDropdown` and `FileNavigator`.

</Column>
<Column size={4} variant={false}>

    <MenuItem active>Item</MenuItem>
    <MenuSeparator />

    <MenuItem>Cool beans!</MenuItem>
  </Menu>
</Story>

</Column>
<Column size={8} variant={false}>

### Action

This variant is primarily used for mixed navigation and selection in dropdowns see AppHeader and DropdownButton.

- `MenuSeparator` will display a single line through content when used in an action menu.

</Column>

  </Menu>
</Story>

</Column>
<Column size={8} variant={false}>

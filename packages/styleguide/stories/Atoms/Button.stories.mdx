We have several versions of buttons that you can use.
Each supports a _light_ and _dark_ mode.

See the [Figma designs](https://www.figma.com/file/N6TfoPI5OA7GanX6Zl8nWj/Buttons-%E2%9C%85?node-id=268%3A0) for a nice grid view of all the variants.
All but CTA button support a size of either `normal` (default) or `small`.

export const buttons = [FillButton, IconButton, StrokeButton, TextButton];
export const styles = ['light', 'light-alt', 'dark', 'dark-alt'];
export const sizes = ['small', 'normal'];

export const ButtonScale = () => {
  const grid = [];
  styles.forEach((mode) => {
    sizes.forEach((size) => {
      buttons.forEach((Component) => {
        const props = { key: `${Component.name}-${mode}-${size}`, mode, size };
        if (Component.name === 'IconButton') {
          return grid.push(
            <Component
              {...props}
              icon={size === 'small' ? MiniDeleteIcon : SearchIcon}
            />
          );
        }
        grid.push(<Component {...props}>{mode}</Component>);
      });
    });
  });
  return (
    <Box
      display="grid"
      alignItems="center"
      justifyItems="start"
      gridTemplateColumns="repeat(4, minmax(100px, max-content))"
      gridAutoRows="3rem"
      columnGap="2rem"
      rowGap="1rem"
      padding="1rem"
      position="relative"
      zIndex={0}
    >
      <Box
        position="absolute"
        left="0"
        right="0"
        bottom="0"
        zIndex={-1}
        top={`${((grid.length / 4) * 4 + 1) / 2}rem`}
        backgroundColor="navy"
      />
      {grid}
    </Box>
  );
};

## Button Variants

We have 2 button color modes: `light` and `dark` which are meant to be used in their respective contexts. For differentiation we also provide an alternate color scheme for lower priority buttons when used adjacently. These are `light-alt` and `dark-alt` respectively.

- **Don't** use a color mode outside of its context (no light buttons on navy backgrounds)
- Use `light` and `dark` modes for your primary buttons.
- Use `light-alt` and `dark-alt` modes for secondary buttons.

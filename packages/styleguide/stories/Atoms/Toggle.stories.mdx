Controlled atom component indicating a user-selectable boolean value.
Receives its `checked` prop from its parent. This is the default theme and size.

## Color Variants

We currently support 2 colors for toggles, this is likely to change. Blue is the default and preferred color especially when adjacent to other form elements.

<Canvas>
  <Story name="blue" args={{ checked: true, variant: 'blue' }}>
    {(args) => <Toggle {...args} />}
  </Story>
  <Story name="hyper" args={{ checked: true, variant: 'hyper' }}>
    {(args) => <Toggle {...args} />}
  </Story>
</Canvas>

## Size

We support 2 sizes for `<Toggle />`.

- `medium`: The default and preferred size, especially if it is part of a form.
- `small`: Used for UI toggles that are outside of a form context and need to fit into smaller areas.

<Canvas>
  <Story name="medium" args={{ size: 'medium' }}>

## Disabled

Disabling the input will change the opacity and cursor but still denote the current value.

  <Story name="disabled" args={{ checked: false, disabled: true }}>

  </Story>
  <Story name="disabled-checked" args={{ checked: true, disabled: true }}>
    {(args) => <Toggle {...args} />}
  </Story>
</Canvas>

## Interactive Example

<Canvas>
  <Story name="interactive" args={{ label: 'interactive' }}>
    {(args) => {
      const [checked, setChecked] = useState(args.checked);
      return (
        <Toggle
          {...args}
          checked={checked}
          onChange={() => setChecked(!checked)}
        />
      );
    }}
  </Story>
</Canvas>

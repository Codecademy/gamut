Badges are generally used as a standard way to highlight a short piece of text, likely a single word. Badges are inline and generally found next to some interface element to be called out.

export const BadgeTemplate = (args) => <Badge {...args}>{args.variant}</Badge>;

## Variants

Badge variants have different meanings between applications. While they are all specified on the `variant` prop not all of them can be used together.

<Canvas>
  <Story name="default" args={{ variant: 'default' }}>
    {BadgeTemplate.bind({})}
  </Story>
  <Story name="stroke" args={{ variant: 'stroke' }}>
    {BadgeTemplate.bind({})}
  </Story>
  <Story name="grey" args={{ variant: 'grey' }}>
    {BadgeTemplate.bind({})}
  </Story>
  <Story name="yellow" args={{ variant: 'yellow' }}>
    {BadgeTemplate.bind({})}
  </Story>
  <Story name="hyper" args={{ variant: 'hyper' }}>
    {BadgeTemplate.bind({})}
  </Story>
  <Story name="blue" args={{ variant: 'blue' }}>
    {BadgeTemplate.bind({})}
  </Story>
</Canvas>

`default` and `stroke` both adapt to their ColorMode (TO-DO)

## States

You can make your badges display with rounded edges using the `round` boolean prop, with the accent font-family using the `accent` prop, or both.

### Round

<Canvas>
  <Story name="round" args={{ variant: 'yellow', round: true }}>
    {BadgeTemplate.bind({})}
  </Story>
</Canvas>

### Stroke

Badges are generally used as a standard way to highlight a short piece of text, likely a single word.
Badges are inline and generally found next to some interface element to be called out.

For example, non-admin Pro users can see a badge calling out Referrals in their dropdown profile menu.

### Variants

Badge variants have different meanings between applications. While they are all specified on the `variant` prop not all of them can be used together. For user facing pages please use only status variants.

#### Status

These variants are for denoting the status of a piece of content without displaying a progress bar or content item icon.

<Canvas>
  <Story name="Product Variants">
    {(args) => (
      <FlexBox gap={16}>
        <Badge variant="new">New</Badge>
        <Badge variant="complete">Complete</Badge>
        <Badge variant="in-progress">In-Progress</Badge>
      </FlexBox>
    )}
  </Story>
</Canvas>

#### Admin

These badge variants for admin pages and should not be used on user facing views.

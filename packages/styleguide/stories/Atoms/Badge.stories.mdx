Badges are generally used as a standard way to highlight a short piece of text, likely a single word. Badges are inline and generally found next to some interface element to be called out.

export const BadgeTemplate = (args) => <Badge {...args}>{args.variant}</Badge>;

### Variants

Badge variants have different meanings between applications. While they are all specified on the `variant` prop not all of them can be used together. For user facing pages please use only status variants.

#### Status

These variants are for denoting the status of a piece of content without displaying a progress bar or content item icon.

<Canvas>
  <Story
    name="status"
    args={{ variant: 'secondary', stroke: true, round: true }}
  >
    {BadgeTemplate.bind({})}
  </Story>
</Canvas>

#### Admin

These badge variants for admin pages and should not be used on user facing views.

### States

You can make your badges display with rounded edges using the `round` boolean prop, with the accent font-family using the `accent` prop, or both.

</Canvas>

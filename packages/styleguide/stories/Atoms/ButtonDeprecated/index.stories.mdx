Primary versions of buttons and links. Previously, all focusable,
interactive elements were using some form of a `ButtonDeprecated` (if
they look like this) or `ButtonDeprecatedBase` (if they have custom
styles).

**Now they're deprecated.**

Prefer `brand-purple` for significant actions such as Pro signups. Prefer `brand-blue` for generic actions.

Play with this button, if you&apos;d like!

## ButtonDeprecatedBase Variants

Use `ButtonDeprecatedBase` when you need something that acts as a link or
button, but don&apos;t need the default button styles.

_Note:_ keep in mind that buttons generally need focus and hover
indicators for accessibility. If you use `ButtonDeprecatedBase`, be sure you&apos;re
adding in your own.

<Canvas>
  <Story name="ButtonDeprecatedBase">
    {(args) => (
      <ButtonDeprecatedBase>
        {text('Label 1', 'I am basic')}
      </ButtonDeprecatedBase>
    )}
  </Story>
  <Story name="ButtonDeprecatedBase Link">
    {(args) => (
      <ButtonDeprecatedBase href="#">

      </ButtonDeprecatedBase>
    )}
  </Story>
  <Story name="ButtonDeprecatedBase Link Style">

      </ButtonDeprecatedBase>

  </Story>
</Canvas>

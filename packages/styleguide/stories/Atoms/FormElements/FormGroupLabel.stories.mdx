### Label htmlFor

It is best to provide `htmlFor` to both the FormGroup + inner form element to make it super accessible. If this is not provided, the FormLabel will default to `<div>`. If an `htmlFor` is provided, it will be a `<label>`.

<Canvas>
  <Story name="FormGroupLabelasLabel">
    {() => (
      <FormGroupLabel htmlFor="label">
        I look like a label, and I am a label.
      </FormGroupLabel>
    )}
  </Story>
  <Story name="FormGroupLabelasDiv">
    {() => (
      <FormGroupLabel>
        I may look like a label, but I'm actually a div.
      </FormGroupLabel>
    )}
  </Story>
</Canvas>

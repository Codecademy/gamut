## Variants

These are our base semantic font scales.

<Canvas>
  <Story name="Variants">
    {(args) => (
      <GridBox
        padding={8}
        rowGap={24}
        columnGap={32}
        gridTemplateColumns="max-content 1fr"
        alignItems="center"
      >
        {Object.keys(typographyStyleVariants).map((variant) => (
          <Fragment key={variant}>
            <Box>
              <Text as="code" fontSize={14}>
                {typographyStyleVariants[variant].fontSize}
              </Text>
            </Box>
            <Box>
              <Text as="span" variant={variant}>
                {variant}
              </Text>
            </Box>
          </Fragment>
        ))}
      </GridBox>
    )}
  </Story>

## Element Preset Variants

When you use a specific element Text will style it with overridable defaults as our best guess behavior. To customize this you may use `variant` and other `typography` system props.

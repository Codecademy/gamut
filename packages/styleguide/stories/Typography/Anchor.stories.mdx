## Variants

We provide 3 variants for links for various use cases

### Interface

The menu link should be used in instances where the context of the link makes it clear that it is clickable. Examples of these contexts include our catalog menu, navigation bar, and footer.

### Standard

For use outside of a ‘menu’ instance where (for a good reason) you can’t use the Text Button with the included button-padding.

### Inline

For use in paragraphs, underlined for extra readability to stand out more beyond just the different color.

export const variants = ['interface', 'standard', 'inline'];

## Usage

- Use this component to navigate between content that is split across multiple pages.

<br />

## Types

We have two types of pagination - `basic` and `includeSkipToButtons`. The `basic` type has number as well as forward and back buttons. The `includeSkipToButtons` type also features an ellipsis button which allows users to quickly jump through page numbers by `chapterSize` as well as the first and last page buttons.

For any pagination that is longer then 10 pages, the `includeSkipToButtons` type is the default.

## Pagination navigation

When using pagination as navigation between linked pages, use the `isNavigation` prop. Your `onChange` function should include programmatic navigation.

<Canvas>
  <Story
    name="As Navigation"
    args={{
      chapterSize: 5,
      totalPages: 27,
      isNavigation: true,
      onChange: (pageNumber) => {
        window.open(
          `https://giphy.com/search/hamtaro-page-${pageNumber}`,
          '_blank'
        );
      },
    }}
  >
    {(args) => <Pagination {...args} />}
  </Story>
</Canvas>

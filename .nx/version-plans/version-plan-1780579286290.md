Remove deprecated `HiddenText` and accordion components

**BREAKING:** `HiddenText` is no longer exported from `@codecademy/gamut`

- Migrate to `<Text screenreader>` for visually hidden, screen-reader-accessible text
- Removed Storybook docs for `HiddenText`; use the `Text` screenreader utility docs instead

**BREAKING:** Remove `AccordionDeprecated`, `AccordionAreaDeprecated`, `AccordionButtonDeprecated`, and associated exported types

- Migrate to `Disclosure` for expandable sections; use `List` with expandable rows when multiple accordions are needed
- Internal `ButtonDeprecated` and `ButtonDeprecatedBase` (accordion-only, not re-exported) are removed with the accordion stack
- Removed Storybook docs for deprecated accordion components
